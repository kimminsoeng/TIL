1. 깃허브란?
   1. Git은 소프트웨어 개발에서 소스 코드를 효과적으로 관리할 수 있도록 하는 무료, 공개 소프트웨어입니다.
   2. Git은 프로젝트 폴더 내에서 작업을 기록하고, 버전 관리를 통해 체계적인 개발이 가능하도록 합니다.
2. 깃허브를 사용하는 이유 
   1. 모든 변경사항을 저장하기 때문에 데이터를 잃어버리지 않는다.
   2. 개발자의 소셜네트워크로 묶인 공간이기 때운에 소스가 버저닝되어 조금 더 좋은 소스로 업데이트 하기 위한 공간이기도 하므로 깃허브를 사용하는 것이 좋다.
3. 깃크라켄 사용법
   1. 깃허브 프로필 화면에서 레포지터리즈를 누른다.
   2. New를 눌러 새 repository를 만든다.
   3. 이름과 설명을 입력하고 Create repository를 누른다.
   4. GitKraken과 Github를 연결한다.(Github에 만들어 둔 저장공간과 사용자의 폴더를 동기화 한다.)
      1. Github 업로드용 폴더를 만든다.
      2. 만들어 둔 폴더를 GitKraken에서 등록해 준다.
      3. 등록작업이 끝나면 등록 된 폴더는 repository로 쓰이게 된다. Gitkraken에서 폴더 아이콘을 누른다.
      4. open a repository를 누른다.
      5. 아까 만들어 둔 폴더를 찾아 선택하고, 폴더 선택을 누른다.
      6. 해당 폴더는 repository가 아니라고 뜨며 repository로 만들겠다고 묻는다. Initialize를 누른다.
      7. 그럼 해당하는 폴더를 repository로 만들기 위한 창이 뜰 것이다. Create repository를 누른다.

4.   이제 Github에 만들어 둔 repository와 사용자 컴퓨터의 repository를 이어 줘야 한다.
   1. 아래 사진처럼 REMOTE 부분에 마우스를 가져다 대면 오른쪽에 Add Remote라는 버트이 생긴다. 그 버튼을 누른다.
   2. Github창에서 URL를 GitKraken에 복붙하고 Add Remote 버튼을 누른다.

5. Pull Requests란 부분에 push버튼을 누른다.
   1. push버튼을 누르면 최상단에 뜨는 창에 submit이라고 쓰여진 문구를 누른다.
